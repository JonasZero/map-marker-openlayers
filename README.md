# OpenLayers 6 map
How to create a simple map (with a marker and polygon) using OpenLayers 6

### Map file include
  - Add marker to map
  - Show map popup on click
  - Create marker on double click
  - Clear all layer markers on double click
  - Create map polygon
  - Test delivery area with polygon
